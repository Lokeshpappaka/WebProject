# Restaurant Website - Simplotel Assignment

This is a responsive restaurant website built with plain HTML and CSS (no frameworks) as per the Simplotel Frontend Developer assignment requirements.

## Features

- ✅ Fully responsive design (Desktop, Tablet, Mobile)
- ✅ Coral-colored header with navigation
- ✅ Gradient background (light yellow to light pink)
- ✅ Responsive hamburger menu for mobile devices
- ✅ Images that don't get pixelated or stretched
- ✅ Works at all zoom levels and window resize
- ✅ No text overflow issues
- ✅ Matches the design specifications

## Setup Instructions

1. Place your burger image file in the root directory and name it `burger.jpg`
   - The image should show a burger with fries and sauces on a plate
   - Recommended dimensions: 1200x800px or similar aspect ratio

2. Open `index.html` in a web browser

## File Structure

```
.
├── index.html      # Main HTML file
├── styles.css      # All CSS styles
├── burger.jpg      # Main image (you need to add this)
└── README.md       # This file
```

## Browser Compatibility

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Responsive Breakpoints

- **Desktop**: > 1024px
- **Tablet**: 768px - 1024px
- **Mobile**: < 768px
- **Small Mobile**: < 480px

## Notes

- All styles are in a single CSS file
- No external libraries or frameworks used
- Pure HTML, CSS, and vanilla JavaScript
- Images use `object-fit: contain` to prevent stretching
- Text uses proper word-wrap to prevent overflow
